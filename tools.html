Free, Private PDF Tools

Every tool on this site follows the same principle: your files never leave your device. Instead of uploading documents to a server for processing, these tools run entirely inside your web browser using open-source JavaScript libraries. That makes them safe for the documents people actually need to work with — bank statements, contracts, identity documents, tax filings, and medical records.

The tools are designed to work together. A typical workflow: convert phone photos to PDF, fix any sideways pages, merge everything into one file, and compress it to meet an upload limit. For step-by-step walkthroughs of common tasks, see our guides.

No signup, no watermarks, no file limits beyond what your device's memory can handle. The site is supported by unobtrusive advertising, which keeps everything free.
